`(backtick) - 백틱 세개를쓰면 박스가 생김
```
<tagname>Contents</tagname>

* 시작 태그만 있는 경우 : Empty Element
```

> 포함관계(Nested)
```
<html>
  <body>
    <h1>Heading</h1>
  </body>
</html>
```

> html-body-h1 관계
> 
> html : body와 관계 - 부모요소 / h1과 관계 - 조상요소
> 
> body : html과 관계 - 자식요소 / h1과 관계 - 부모요소
> 
> h1 : body와 관계 - 자식요소 / html과 관계 - 자손요소
